Bash Cheatsheet

Navigation:

  • pwd: Print working directory.
  • cd <directory>: Change directory. cd ..: Go to parent directory. cd ~: Go to home directory. cd -: Go to previous directory.
  • ls: List directory contents. ls -l: Long listing (details). ls -a: Show hidden files. ls -lh: Human-readable sizes.
  • mkdir <directory>: Create a directory.
  • rmdir <directory>: Remove an empty directory.
  • rm <file>: Remove a file. rm -r <directory>: Remove a directory recursively (use with caution!). rm -f <file>: Force removal without confirmation.
  • cp <source> <destination>: Copy a file or directory. cp -r <source> <destination>: Recursively copy a directory.
  • mv <source> <destination>: Move or rename a file or directory.
  • find <path> <options>: Find files. Example: find . -name "*.txt" finds all .txt files in the current directory and subdirectories.

File Manipulation:

  • cat <file>: Concatenate and display files.
  • less <file>: View a file page by page. /pattern: Search within less. n: Next match. N: Previous match. q: Quit.
  • head <file>: Display the first few lines of a file (default 10). head -n 20 <file>: Display the first 20 lines.
  • tail <file>: Display the last few lines of a file (default 10). tail -f <file>: Follow the file and display new lines as they are added.
  • wc <file>: Word count. wc -l <file>: Line count. wc -w <file>: Word count.
  • grep <pattern> <file>: Search for a pattern in a file. grep -i <pattern> <file>: Case-insensitive search. grep -r <pattern> <directory>: Recursive search.
  • sed <command> <file>: Stream editor for text transformations. Example: sed 's/old/new/g' <file> replaces “old” with “new” globally.
  • awk '{print $1}' <file>: Process text files line by line. Example: prints the first field of each line.

Input/Output Redirection:

  • >: Redirect output to a file. command > output.txt
  • >>: Append output to a file. command >> output.txt
  • <: Redirect input from a file. command < input.txt
  • |: Pipe the output of one command to the input of another. command1 | command2

Variables:

  • variable_name="value": Assign a value to a variable.
  • ${variable_name}: Access the value of a variable.
  • echo $variable_name: Print the value of a variable.

Control Flow:

  • if [ condition ]; then ...; fi: Conditional statement.
  • for i in {1..10}; do ...; done: Loop.
  • while [ condition ]; do ...; done: Loop.

Other Useful Commands:

  • echo <text>: Print text to the console.
  • date: Display the current date and time.
  • cal: Display a calendar.
  • man <command>: Display the manual page for a command.
  • help <command>: Display brief help for a command.
  • sudo <command>: Execute a command with root privileges. (Use with caution!)
  • history: Display command history. !n: Execute command number n from history.
  • clear: Clear the console screen.
  • kill <process_id>: Terminate a process. pkill <process_name>: Kill processes by name.
